Hilfe

Neues Spiel beginnen

Um ein neues Spiel zu beginnen, geben Sie rechts Ihren Beutzernamen in das Eingabefeld "Benutzername" ein und wählen Sie die Farbe ihrer Spielfigur, bevor Sie auf "Neues Spiel eröffnen" klicken. Wenn Sie ein Spiel eröffnen, sind Sie zugleich der Administrator des Spiels, d.h. Sie haben das Recht, das Spiel zu starten, sobald andere Spieler beigetreten sind.

Spiel beitreten

Um einem Spiel beizutreten, geben Sie links Ihren Beutzernamen in das Eingabefeld "Benutzername" ein und wählen Sie die Farbe ihrer Spielfigur, bevor Sie ein laufendes Spiel aus der Liste auswählen und auf "Spiel beitreten" klicken.

Spielen

Nachdem der Administrator auf "Spiel starten" geklickt hat; muss der Spieler, der gerade an der Reihe ist, würfeln; dies geschieht durch einen Klick auf den Button "Würfeln". Die Würfelzahl wird auf dem Dialogfeld in der Mitte angezeigt und die Spielfigur um diese Zahl weitergerückt. Nach dem Würfeln kann man sich auf folgenden Feldern befinden:

Straßen / Bahnhöfe / Werke

Straßen/Bahnhöfe/Werke können gekauft werden; bei Straßen besteht, sofern Sie diese bereits erstanden haben, zusätzlich die Möglichkeit zum Bauen. Je nach gewählter Option wird ein entsprechender Betrag vom Konto abgezogen. Ist die Straße bereits verkauft; wird dies auf der Dialogbox in der Mitte des Spielfeldes angezeigt und ein entsprechender Mietbetrag vom Konto abgezogen. Wenn sie eine Straße nicht kaufen wollen oder den Kauf bzw. Bau abgeschlossen haben, klicken Sie auf "Zug beenden". Der Nächste Spieler ist dran. Weiterhin können Sie mit einem Klick auf eine Straße feststellen, wem diese gehört. Ihre Käufe können Sie auf der Liste rechts einsehen.

Ereignis-/ Gemeinschaftsfelder

Bei einem Ereignis- beziehungsweise Gemeinschaftsfeld wird ein Ereignis auf der Dialogbox ausgegeben, was entsprechende Auswirkungen auf Ihren Kontostand hat.
